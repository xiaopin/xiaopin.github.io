越来越多人采用Markdown进行写作，而表格作为一种比较常见的需求，Markdown也支持快速创建表格。

语法:
```
| Name | Score |
| :- | :-: |
| zhangsan | 90 |
| lisi | 100 |
```

效果:

| Name | Score |
| :-: | :-: |
| zhangsan | 90 |
| lisi | 100 |

语法说明:
- 第一行为表头，第二行为分割表头和表格内容，第三行开始每一行表示一个表格行
- 列与列之间用`|`进行分割
- 第二行中可以指定内容对其模式
  - `-` 默认对其
  - `:-` 左对齐
  - `-:` 右对齐
  - `:-:` 居中对齐
